🏰 新版正方教务爬虫 For PHP 
===============
基于 swoole 的正方教务爬虫

## Requirement

- [PHP 7.1 +](http://php.net/manual/zh/migration71.new-features.php)
- [Swoole](https://www.swoole.com)
- [easySwoole 3.x](http://easyswoole.com/)
- [Composer 1.0](http://pkg.phpcomposer.com/)

## Usage

```php
$ composer install 
$ php easyswoole start
```
